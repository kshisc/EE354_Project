# Dragon Game
This project implements the traditional Snake arcade game on a Nexys A7 FPGA board, using Verilog. The player controls a dragon sprite that moves across the VGA display, consuming leaves, gems, and birds to grow longer while avoiding collisions with the boundaries and its own body. As the dragon gets longer and “levels up,” the playable area shrinks and the dragon’s speed increases, making the game more challenging.
